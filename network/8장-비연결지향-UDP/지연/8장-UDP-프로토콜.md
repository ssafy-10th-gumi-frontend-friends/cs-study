# 8장 UDP 프로토콜

24.03.04  
사용자 데이터그램 프로토콜 User Datagram protocol  
유니버설 데이터그램 프로토콜이라고 일컫기도 한다.

UDP의 전송방식은 너무 단순해서 서비스의 **신뢰성이 낮고**, 데이터그램 도착 순서가 바뀌거나, 중복되거나, 심지어는 통보없이 누락시키기도 한다.

UDP는 일반적으로 오류의 검사와 수정이 필요없는 프로그램에서 수행할 것으로 가정한다.
